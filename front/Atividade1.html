Projeto Escolar: Bem Vindo(a)

O projeto tem como função ajudar a resolver problemas escolares com facilidade e eficiência.

Aviso para pais

Esse problema foi escolhido pois a escola não faz avisos exclusivamente para os pais, dando para os alunos e confiando que eles iram dar o aviso para os pais, mas tem varios que perdem esses avisos ou esquece de entregar.

Feedback para os que utilizam o aplicativo

Isso vai ser uma página dedicada para ver se estão gostando ou achando ruim o site.

Funcionalidades:

    O sistema deve permitir o cadastro de usuários;
    O sistema deve permitir o login de usuários;
    O sistema deve validar email e senha no login;
    O sistema deve permitir visualizar usuários cadastrados;
    O sistema deve permitir criar eventos;
